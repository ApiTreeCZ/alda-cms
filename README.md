# ALDA CMS

> CMS web system

## System obsahuje

-   Verejna cast
-   Administrace

## Predpoklady pro vyvoj

-   Node.js (10 a vyssi)
-   Git

## Instalace

```npm
npm install
```

## Skripty

**Dev rezim:**

```npm
npm run dev
```

**Produkce:**

```npm
npm run build
```

a pote

```npm
npm run start
```
